# 22. Roadmap V2.0+

## V2.1 (Corto plazo)

- Exponer detalle público de writeups por slug
- Dashboard público de métricas agregadas (sin datos sensibles)
- Mejora de observabilidad (tracing y métricas)

## V2.2 (Medio plazo)

- Multiusuario real (sin `default-user` fallback)
- RBAC granular por recurso
- Audit trails exportables

## V2.3 (Medio plazo)

- Integración CI con escaneo SAST/Dependency
- Entorno staging con despliegue automático
- Health checks extendidos y readiness checks

## V3.0 (Largo plazo)

- Tenant por organización
- Integración de evidencias firmadas
- Sistema de badges académicos verificables
