# 02. Casos de Uso

## Actores
- Usuario Owner
- Usuario Student
- Sistema

## Casos principales
1. Iniciar sesion en plataforma privada.
2. Registrar sesion de estudio.
3. Crear/editar nota tecnica en Knowledge Hub.
4. Registrar progreso de curso/modulo/leccion.
5. Registrar laboratorio y evidencia.
6. Crear writeup tecnico en Markdown/MDX.
7. Actualizar avance de certificacion.
8. Gestionar proyecto y prioridad.
9. Ejecutar herramienta defensiva (ej. DNS Lookup).
10. Simular evento SIEM y revisar regla.
11. Registrar activo de home lab.
12. Consultar dashboard ejecutivo.

## Flujo resumido de alto valor
- Login -> Dashboard -> modulo especifico -> registro/edicion -> auditoria -> actualizacion de metricas.
