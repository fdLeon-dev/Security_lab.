# 20. Manual de Usuario

## 1. Acceso

### Portafolio público

1. Abrir la raíz del sistema
2. Seleccionar `Ver Portafolio`
3. Explorar proyectos, certificaciones, tecnologías y timeline

### Plataforma privada

1. Abrir `Acceso Privado`
2. Ingresar credenciales
3. Navegar los módulos internos

## 2. Módulos privados

- Dashboard
- Knowledge Hub
- Learning Tracker
- Lab Manager
- Writeup Center
- Certification Tracker
- Projects
- Toolkit defensivo
- SIEM Academy
- Home Lab Inventory

## 3. Buenas prácticas

- No compartir credenciales
- Utilizar contraseñas robustas
- Revisar historial y auditoría en sesiones de demo
- Mantener writeups sensibles en modo PRIVATE/INTERNAL

## 4. Flujo para presentación profesional

1. Entrar a `/portfolio`
2. Presentar tecnologías dominadas
3. Mostrar timeline de evolución
4. En demo técnica, entrar a modo privado para evidencias internas
